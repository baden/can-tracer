# План

Треба створити WEB-додаток.
Використати наступний набір:

- пакетний менеджер: npm
- мова: vanila js
- збірка: vite

Мінімалістичний сучасний інтерфейс. Використовувати розумно простір, зробити елементи інтерфейсу з можливістю розподіляти простір.


## Задача для додатка.

Працювати з USB-CAN перетворювачем по протоколу SLCAN через COM-порт.

1. Реалізувати налантування параметрів CAN-шини: FD/Normal, швидкисть (враховуючи FD), std/ext, listen only, etc.
2. Реалізувати непреривне логування CAN-шини. Писати в буфер.
3. Реалізувати можливість відправки в CAN-шину пакету. Зробити поля для вводу ID, розміру, даних і т.д. Реалізувати можливість автоматичної періодичної передачі пакету.
4. Реалізувати можливість записувати трейси в файл.
5. Реалізувати можливість відтворювати трейси з файлу. Однократно, та періодично, з вказанням інтервалу.

## Безпереривне логування

Логування треба реалізувати у пам'ять, так, шоб воно могло витримувати велику інтенсивність даних, і не призводило к надмірному навантаженню на оновлення інтерфейсу. Писати отремо, а показувати результат окремо. Реалізувати відображення поточної статистики логування, скільки пакетів отримано, а сам просмотр логів реалізувати якось розумно. Показувати останні отримані пакети можно і безпереривно, не не повинно черезмірно навантажити інтерфейс.

Реалізувати кнопки очищення, збереження в файл. Показувати мітки часу при отриманні пакетів, а також час, якйй пройшов після отримання попереднього пакету (час затримки між пакетами).
